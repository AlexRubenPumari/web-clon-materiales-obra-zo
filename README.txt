## Descripción

Este proyecto consiste en un clon del sitio web Maderas Kopruch, tiene como objetivo mejorar mis habilidades en HTML y CSS. Se ha aplicado la metodología BEM para la nomenclatura de clases CSS, lo cual facilita el mantenimiento y la escalabilidad del código.
